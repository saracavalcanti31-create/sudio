FAQ

Perguntas frequentes

O que é o Nexus7?+

Nexus7 é uma plataforma de criação com inteligência artificial para makers e empreendedores. Permite gerar artes, criativos para redes sociais, carrosseis e identidade visual profissional sem precisar de designer ou conhecimento técnico.

Preciso saber usar design para usar o Nexus7?+

Não. O Nexus7 foi criado para quem não é designer. A plataforma usa IA e templates prontos para gerar criativos profissionais a partir de um simples prompt de texto. Há também onboarding guiado e vídeos passo a passo para novos usuários.

Quais são os planos disponíveis?+

O Nexus7 oferece três planos: Essencial (20 gerações/dia, templates, exportação HD), Creator (gerações ilimitadas, API própria, todos os templates e suporte prioritário) e Studio (tudo do Creator + 5 usuários, projetos compartilhados e gestão de marca).

O Nexus7 tem garantia de reembolso?+

Sim. O Nexus7 oferece garantia de 7 dias. Caso não fique satisfeito com a plataforma, o reembolso de 100% é feito sem perguntas.

Posso usar minha própria chave de API?+

Sim. Os planos Creator e Studio permitem conectar sua própria chave de API para gerar sem limites, pagando apenas pelo que consumir diretamente no provedor de IA escolhido.
